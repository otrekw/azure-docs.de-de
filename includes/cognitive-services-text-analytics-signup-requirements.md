Schlüssel und Endpunkt für eine Textanalyseressource Azure Cognitive Services werden von Azure-Ressourcen dargestellt, die Sie abonnieren. Erstellen Sie mithilfe des [Azure-Portals](../articles/cognitive-services/cognitive-services-apis-create-account.md) oder der [Azure CLI](../articles/cognitive-services/cognitive-services-apis-create-account-cli.md) auf Ihrem lokalen Computer eine Ressource für die Textanalyse. Sie können Ihre Ressource auch im [Azure-Portal](https://portal.azure.com/) anzeigen.
