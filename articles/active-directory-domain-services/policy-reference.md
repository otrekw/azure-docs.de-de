# <a name="azure-policy-built-in-definitions-for-azure-active-directory-domain-services"></a>Integrierte Richtliniendefinitionen von Azure Policy für Azure Active Directory Domain Services

Diese Seite bietet einen Index der in [Azure Policy](../governance/policy/overview.md) integrierten Richtliniendefinitionen für Azure Active Directory Domain Services. Weitere Azure Policy-Integrationen für andere Dienste finden Sie unter [Integrierte Azure Policy-Richtliniendefinitionen](../governance/policy/samples/built-in-policies.md).

Die Namen der einzelnen integrierten Richtliniendefinitionen sind Links zur entsprechenden Richtliniendefinition im Azure-Portal. Verwenden Sie den Link in der Spalte **Version**, um die Quelle im [Azure Policy-GitHub-Repository](https://github.com/Azure/azure-policy) anzuzeigen.
