# <a name="azure-synapse-analytics-data-integration-partners"></a>Partner für die Azure Synapse Analytics-Datenintegration

Um Ihre Data Warehouse-Lösung mit dem Feature „Synapse SQL-Pool“ von Azure Synapse zu erstellen, steht Ihnen eine große Auswahl branchenführender Tools zur Verfügung. In diesem Artikel sind Partnerunternehmen von Microsoft aufgeführt, die offizielle Tools und Lösungen für die Datenintegration anbieten, die Azure Synapse unterstützen.
